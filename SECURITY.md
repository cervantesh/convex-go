# Security Policy

This is a community Go client for Convex. Do not file public issues for
secrets, credential leaks, or exploitable vulnerabilities.

## Reporting

If GitHub private vulnerability reporting is enabled for this repository, use
that channel.

If no private channel is available, open a minimal public issue that says only:
"Security report requested." Do not include exploit details, secrets, stack
traces with tokens, deployment URLs, or private user data in the public issue.
A maintainer will arrange a private handoff before details are shared.

## Supported Versions

Only the latest pre-v1 release line is expected to receive fixes.
