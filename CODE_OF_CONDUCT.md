# Code of Conduct

This is a project-specific conduct policy for the convex-go community.

## Our Standard

Be respectful, practical, and specific. Focus disagreement on the code,
behavior, documentation, or compatibility claim being discussed.

## Enforcement

Maintainers may moderate or remove comments, issues, and pull requests that are
abusive, harassing, discriminatory, spammy, or unrelated to the project.

If you need to report a conduct issue privately and no private project contact
is available, open a minimal public issue requesting maintainer contact without
including private details.
